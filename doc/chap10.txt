  [1X10.1 [33X[0;0YBol loops and Bruck loops[133X[101X
  
  [33X[0;0YA  right  Bruck  loop is a right Bol loop satisfying the automorphic inverse
  property. If [23XQ[123X is a right Bol loop in which the squaring map [23Xx\mapsto x^2[123X is
  a   permutation,   then   [23X(Q,\circ)[123X   with   multiplication   [23Xx\circ   y   =
  ((x(yy))x)^{1/2}[123X is a right Bruck loop, the right Bruck loop [13Xassociated[113X with
  [23XQ[123X.  Here,  [23Xx^{1/2}[123X  is  the square-root of [23Xx[123X, the unique element [23Xy[123X of [23XQ[123X such
  that [23Xy^2=x[123X.[133X
  
  [33X[0;0YFor  a left Bol loop in which squaring is a permutation, the associated left
  Bruck loop is defined by [23Xx\circ y = (x((yy)x))^{1/2}[123X.[133X
  
  
  [1X10.1-1 [33X[0;0YAssociated Bruck loop[133X[101X
  
  [33X[1;0Y[29X[2XAssociatedRightBruckLoop[102X( [3XQ[103X ) [32X attribute[133X

  [6XReturns:[106X  [33X[0;10Ythe  right  (left) Bruck loop associated with the right (left) Bol
            loop [10XQ[110X. The returned loop is index based iff [10XQ[110X is index based.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XQ := RightBolLoop(21,1);;[127X[104X
    [4X[25Xgap>[125X [27XAssociatedRightBruckLoop( Q );[127X[104X

  [1X10.2 [33X[0;0YTriality for Moufang loops[133X[101X
  
  [33X[0;0YLet  [23XG[123X be a group and [23X\sigma[123X, [23X\rho[123X be automorphisms of [23XG[123X satisfying [23X\sigma^2
  =  \rho^3  =  (\sigma \rho)^2 = 1[123X. Below we write automorphisms as exponents
  and [23X[g,\sigma][123X for [23Xg^{-1}g^\sigma[123X. We say that the triple [23X(G,\rho,\sigma)[123X is
  a  [13Xgroup with triality[113X if [23X[g, \sigma] [g,\sigma]^\rho [g,\sigma]^{\rho^2} =1[123X
  holds  for  all  [23Xg  \in  G[123X.  It is known that one can associate a group with
  triality  [23X(G,\rho,\sigma)[123X  in  a  canonical  way  with a Moufang loop [23XQ[123X. See
  [NVe03] for more details.[133X
  
  [33X[0;0YFor any Moufang loop [23XQ[123X, we can calculate the triality group as a permutation
  group acting on [23X3|Q|[123X points. If the multiplication group of [23XQ[123X is polycyclic,
  then  we can also represent the triality group as a pc group. In both cases,
  the  automorphisms [23X\sigma[123X and [23X\rho[123X are in the same family as the elements of
  [23XG[123X.[133X
  
  [1X10.2-1 TrialityPermGroup[101X
  
  [33X[1;0Y[29X[2XTrialityPermGroup[102X( [3XQ[103X ) [32X operation[133X
  [6XReturns:[106X  [33X[0;10Ya  record  with components [10XG[110X, [10Xrho[110X, [10Xsigma[110X, where [10XG[110X is the canonical
            group  with  triality  associated  with a Moufang loop [3XQ[103X, and [10Xrho[110X,
            [10Xsigma[110X  are the corresponding triality automorphisms. We allow [3XQ[103X to
            be a group.[133X
  
  [1X10.2-2 TrialityPcGroup[101X
  
  [33X[1;0Y[29X[2XTrialityPcGroup[102X( [3XQ[103X ) [32X operation[133X
  
  [33X[0;0YThis  is  a  variation  of  [10XTrialityPermGroup[110X in which [10XG[110X is returned as a pc

  [1X10.3 [33X[0;0YQuarter modifications of Moufang loops[133X[101X
  
  [33X[0;0YDrÃ¡pal  [Dr\03]  described  two  prominent families of extensions of Moufang
  loops.   The  common  feature  of  these  extensions  is  that  they  modify
  multiplication  tables  of  groups  or  other  Moufang  loops in exactly one
  quarter  of  the  mutliplication  table.  It turns out that these extensions
  suffice  to  obtain  all nonassociative Moufang loops of order at most 64 if
  one starts with so-called Chein loops. We call the two constructions [13XMoufang
  modifications[113X.  The  library  of  Moufang  loops used to be based on Moufang
  modifications. See [DVe06] for details.[133X
  
  [1X10.3-1 CheinLoop[101X
  
  [33X[1;0Y[29X[2XCheinLoop[102X( [3XG[103X[, [3XconstructorStyle[103X] ) [32X operation[133X
  [6XReturns:[106X  [33X[0;10Ythe  Chein loop constructed from the group [10XG[110X, that is, the Moufang
            loop on [10X[0,1] x G[110X with multiplication[133X
  
  
  [24X          [33X[0;10Y(0,x)(0,y)   =   (0,xy),\  (0,x)(1,y)  =  (1,yx),\,  (1,x)(0,y)  =
            (1,xy^{-1}),\ (1,x)(1,y) = (0,yx^{-1}).[133X
  
  [124X
  
            [33X[0;10YThe resulting Moufang loop is associative iff [10XG[110X is commutative.[133X
