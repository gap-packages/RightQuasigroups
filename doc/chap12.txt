  
  [1X12 [33X[0;0YLibraries of loops, racks and quandles[133X[101X
  
  [33X[0;0YLibraries  of  small  loops,  racks  an  quandles  form  an integral part of
  [5XRightQuasigroups[105X.  The  algebras  are  stored in libraries up to isomorphism
  and, occasionally, up to isotopism.[133X
  
  
  [1X12.1 [33X[0;0Y*A typical library[133X[101X
  
  [33X[0;0YA  library  named [13Xmy algebras[113X is stored in file [11Xdata/my_algebras.tbl[111X and the
  corresponding  data structure is named [10XRQ_my_algebras[110X. For example, when the
  library  is  called  [13Xleft  Bol  loops[113X, the corresponding data file is called
  [11Xdata/left_Bol_loops.tbl[111X  and  the  corresponding  data  structure  is  named
  [10XRQ_left_Bol_loops[110X.[133X
  
  [33X[0;0YIn most cases, the array [10XRQ_my_algebras[110X consists of three lists:[133X
  
  [30X    [33X[0;6Y[10XRQ_my_algebras[1][110X  is a list of orders for which there is at least one
        algebra in the library,[133X
  
  [30X    [33X[0;6Y[10XRQ_my_algebras[2,k][110X    is    the   number   of   algebras   of   order
        [10XRQ_my_algebras[1,k][110X in the library,[133X
  
  [30X    [33X[0;6Y[10XRQ_my_algebras[3][k][s][110X  contains  data  necessary  to produce the [10Xs[110Xth
        algebra of order [10XRQ_my_algebras[1,k][110X in the library.[133X
  
  [33X[0;0YThe  format  of  [10XRQ_my_algebras[3][110X depends heavily on the particular library
  and is not standardized in any way. The data is often coded to save space.[133X
  
  [33X[0;0YThe library names are, in alphabetical order:[133X
  
  [30X    [33X[0;6Y"automorphic loops",[133X
  
  [30X    [33X[0;6Y"CC loops",[133X
  
  [30X    [33X[0;6Y"connected quandles",[133X
  
  [30X    [33X[0;6Y"code loops",[133X
  
  [30X    [33X[0;6Y"interesting loops",[133X
  
  [30X    [33X[0;6Y"itp small loops",[133X
  
  [30X    [33X[0;6Y"LCC loops",[133X
  
  [30X    [33X[0;6Y"left Bol loops",[133X
  
  [30X    [33X[0;6Y"left Bruck loops",[133X
  
  [30X    [33X[0;6Y"Moufang loops",[133X
  
  [30X    [33X[0;6Y"nilpotent loops",[133X
  
  [30X    [33X[0;6Y"Paige loops",[133X
  
  [30X    [33X[0;6Y"right Bol loops",[133X
  
  [30X    [33X[0;6Y"right Bruck loops",[133X
  
  [30X    [33X[0;6Y"RCC loops",[133X
  
  [30X    [33X[0;6Y"small loops",[133X
  
  [30X    [33X[0;6Y"small quandles",[133X
  
  [30X    [33X[0;6Y"small racks",[133X
  
  [30X    [33X[0;6Y"Steiner loops".[133X

  [1X12.2-3 LibraryAlgebras[101X
  
  [33X[1;0Y[29X[2XLibraryAlgebras[102X( [3Xname[103X, [3Xfilters[103X ) [32X function[133X
  [6XReturns:[106X  [33X[0;10Yall  algebras  from  the  library  named [3Xname[103X that satisfy all the
            conditons specified by [3Xfilters[103X. The argument list [3Xfilters[103X consists
            of  alternating  functions  and accepted values of those functions
            (e.g.,  [10XSize,  [3..6],  IsCommutative,  false[110X), with the following
            conventions:[133X
  
            [30X    [33X[0;16Ythe  first  function is understood to be [10XSize[110X, whether it is
                  given or not,[133X
  
            [30X    [33X[0;16Yif the given accepted values [10Xv[110X for a function [10Xf[110X form a list,
                  then  an  algebra [10XA[110X passes the condition on [10Xf[110X iff [10Xf(A)[110X is in
                  [10Xv[110X,[133X
  
            [30X    [33X[0;16Yif  the given accepted values [10Xv[110X for a function [10Xf[110X do not form
                  a list, then an algebra [10XA[110X passes the condition on [10Xf[110X iff [10Xf(A)[110X
                  is equal to [10Xv[110X,[133X
  
            [30X    [33X[0;16Ythe accepted values for [10XSize[110X must always be given,[133X
  
            [30X    [33X[0;16Yif  the  accepted  values for a function [10Xf[110X (other than [10XSize[110X)
                  are omitted, then an algebra [10XA[110X passes the condition on [10Xf[110X iff
                  [10Xf(A)[110X is [10Xtrue[110X.[133X
  
            [33X[0;10YNote  that  the  function  delivers  all  algebras  IN THE LIBRARY
            satisfying   the  conditions,  not  all  algebras  satisfying  the
            conditions.  For  instance,  requesting all Moufang loops of order
            2048 will deliver an empty list because there are no such loops in
            the  library,  although there certainly are nonassociative Moufang
            loops  of  order  2048.  Care  must  be  taken  especially  in the
            situations when the library has gaps in the implemented orders.[133X
  
  [33X[0;0YIt   is   aso  possible  to  access  individual  libraries  directly,  e.g.,
  [10XMoufangLoops( [110X[3Xfilters[103X[10X )[110X.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XLibraryAlgebras( "Moufang loops", Size, 81, IsCommutative, false );[127X[104X
    [4X[28X[ MoufangLoop( 81, 3 ), MoufangLoop( 81, 4 ), MoufangLoop( 81, 5 ) ][128X[104X
    [4X[25Xgap>[125X [27XMoufangLoops( [65..81], IsCommutative, Exponent, 3 ); [127X[104X
    [4X[28X[ MoufangLoop( 81, 1 ) ][128X[104X
  [4X[32X[104X
  
  
  [1X12.3 [33X[0;0YLibrary of left Bol loops and right Bol loops[133X[101X
  
  [33X[0;0YThe  library named [13Xleft Bol loops[113X contains all nonassociative left Bol loops
  of  order  less  than  17,  including Moufang loops, as well as all left Bol
  loops  of order [23Xpq[123X for primes [23Xp>q>2[123X. There are 6 such loops of order 8, 1 of
  order 12, 2 of order 15, 2038 of order 16, and [23X(p+q-4)/2[123X of order [23Xpq[123X.[133X
  
  [33X[0;0YThe  classification  of left Bol loops of order 16 was first accomplished by
  Moorhouse  [Moo]. Our library was generated independently and it agrees with
  Moorhouse's  results.  The  left  Bol  loops  of order [23Xpq[123X were classified in
  [KNVe15].[133X
  
  [33X[0;0YNote:  Only  left  Bol  loops are stored in the library. Right Bol loops are
  retrieved by calling [10XOppositeLoop[110X on left Bol loops.[133X

  [1X12.4 [33X[0;0YLibrary of left Bruck loops and right Bruck loops[133X[101X
  
  [33X[0;0YThe  library  named [13Xleft Bruck loops[113X contains all left Bruck loops of orders
  3, 9, 27 and 81 (there are 1, 2, 7 and 72 such loops, respectively).[133X
  
  [33X[0;0YFor  an  odd  prime [23Xp[123X, left Bruck loops of order [23Xp^k[123X are centrally nilpotent
  and  hence central extensions of the cyclic group of order [23Xp[123X by a left Bruck
  loop  of order [23Xp^{k-1}[123X. It is known that left Bruck loops of order [23Xp[123X and [23Xp^2[123X
  are  abelian  groups;  we  have  included them in the library because of the
  iterative nature of the construction of nilpotent loops.[133X

  [1X12.8 [33X[0;0YLibrary of conjugacy closed loops[133X[101X
  
  [33X[0;0YThe  library  named  [13XRCC  loops[113X  contains  all nonassocitive right conjugacy
  closed  loops  of  order [23Xn\le 27[123X up to isomorphism. The data for the library
  was  generated  by  Katharina  Artic [Art15] who can also provide additional
  data for all right conjugacy closed loops of order [23Xn\le 31[123X.[133X
  
  [33X[0;0YLet [23XQ[123X be a right conjugacy closed loop, [23XG[123X its right multiplication group and
  [23XT[123X  its right section. Then [23X\langle T\rangle = G[123X is a transitive group, and [23XT[123X
  is  a  union of conjugacy classes of [23XG[123X. Every right conjugacy closed loop of
  order [23Xn[123X can therefore be represented as a union of certain conjugacy classes
  of  a transitive group of degree [23Xn[123X. This is how right conjugacy closed loops
  of order less than [23X28[123X are represented in [5XRightQuasigroups[105X.[133X
  
  [33X[0;0YThe following table summarizes the number of right conjugacy closed loops of
  a given order up to isomorphism:[133X
  
  
  [24X[33X[0;6Y\begin{array}{r|rrrrrrrrrrrrrr}  order  &6&  8&9&10&  12&14&15&  16\cr loops
  &3&19&5&16&155&97& 17&6317\cr \hline order &18& 20&21& 22& 24& 25& 26& 27\cr
  loops &1901&8248&119&10487&471995& 119&151971&152701 \end{array}[133X
  
  [124X
  
  [33X[0;0YNote:  Only the right conjugacy closed loops are stored in the library. Left
  conjugacy  closed  loops  are obtained from right conjugacy closed loops via
  [10XOppositeLoop[110X.[133X

  [1XRightConjugacyClosedLoop[133X[101X

  [33X[1;0Y[29X[2XRightConjugacyClosedLoop[102X( [3Xn[103X, [3Xm[103X ) [32X operation[133X
  [6XReturns:[106X  [33X[0;10Ythe  [3Xm[103Xth  left conjugacy closed loop (right conjugacy closed loop)
            of order [3Xn[103X.[133X

  [6XReturns:[106X  [33X[0;10Yall  left  conjugacy  closed  loops (right conjugacy closed loops)
            satisfying    the    conditions    specified   by   [3Xfilters[103X.   See
            [10XLibraryAlgebras[110X above for the conventions on [3Xfilters[103X.[133X
  
  [33X[0;0YThe  library  named  [13XCC loops[113X contains all CC loops of order [23X2\le 2^k\le 64[123X,
  [23X3\le  3^k\le  81[123X,  [23X5\le  5^k\le  125[123X, [23X7\le 7^k\le 343[123X, all nonassociative CC
  loops  of  order  less than 28, and all nonassociative CC loops of order [23Xp^2[123X
  and [23X2p[123X for any odd prime [23Xp[123X.[133X
  
  [33X[0;0YBy  results  of  Kunen  [Kun00], for every odd prime [23Xp[123X there are precisely 3
  nonassociative  conjugacy  closed  loops  of  order  [23Xp^2[123X.  Csörgő and Drápal
  [CoD05]   described   these   3   loops   by   multiplicative   formulas  on
  [23X\mathbb{Z}_{p^2}[123X and [23X\mathbb{Z}_p \times \mathbb{Z}_p[123X as follows:[133X
  
  [30X    [33X[0;6YCase  [23Xm = 1[123X:Let [23Xk[123X be the smallest positive integer relatively prime to
        [23Xp[123X  and  such  that  [23Xk[123X  is  a  square  modulo  [23Xp[123X  (i.e.,  [23Xk=1[123X).  Define
        multiplication on [23X\mathbb{Z}_{p^2}[123X by [23Xx\cdot y = x + y + kpx^2y[123X.[133X
  
  [30X    [33X[0;6YCase [23Xm = 2[123X: Let [23Xk[123X be the smallest positive integer relatively prime to
        [23Xp[123X  and  such that [23Xk[123X is not a square modulo [23Xp[123X. Define multiplication on
        [23X\mathbb{Z}_{p^2}[123X by [23Xx\cdot y = x + y + kpx^2y[123X.[133X
  
  [30X    [33X[0;6YCase  [23Xm = 3[123X: Define multiplication on [23X\mathbb{Z}_p \times \mathbb{Z}_p[123X
        by [23X(x,a)(y,b) = (x+y, a+b+x^2y )[123X.[133X
  
  [33X[0;0YMoreover,  Wilson  [WJ75] constructed a nonassociative conjugacy closed loop
  of  order  [23X2p[123X for every odd prime [23Xp[123X, and Kunen [Kun00] showed that there are
  no  other  nonassociative  conjugacy  closed oops of this order. Here is the
  relevant   multiplication   formula  on  [23X\mathbb{Z}_2  \times  \mathbb{Z}_p[123X:
  [23X(0,m)(0,n) = ( 0, m + n )[123X, [23X(0,m)(1,n) = ( 1, -m + n )[123X, [23X(1,m)(0,n) = ( 1, m +
  n)[123X, [23X(1,m)(1,n) = ( 0, 1 - m + n )[123X.[133X
